Browser-Einstellungen

Mustervergleich

Der Mustervergleich ermöglicht es Ihnen eine vordefinierte URL automatisch in einer anderen Anwendung zu öffnen, JavaScript und UserAgent zu ändern oder Sperren zu setzen
Um einen Mustervergleich zu erstellen, beachten Sie bitte Spezifikation Mustervergleich.

UserScript-Einstellungen

Dort kann man UserScripte erstellen und verwalten

Werbeblocker-Einstellungen

Sie können Werbeblocker einrichten.
Informationen zum Schreiben der jeweiligen Liste finden Sie unter Spezifikation Mustervergleich.

Ressourcen-Interrupt-Einstellungen

Überprüft die URL und blockiert Bilder, externe Skripte, etc.
Es kann auch für verwendet werden, um Werbung zu blockieren.

Startseite

öffnet die angegebene URL beim Öffnen eines neuen Reiters etc.
Die Voreinstellung ist Schnellwahl (yuzu:speeddial).

Download Aktion

Sie können das Verhalten beim Herunterladen von Dateien usw. ändern.

Nichts unternehmen
Der Download-Request wird ignoriert.
Auto
Im Falle einer Videowiedergabe-Anforderung oder dergleichen wird die URL von einer anderen Anwendung geöffnet, ansonsten wird ein Download-Dialog angezeigt.
Immer herunterladen
Immer den Download-Dialog anzeigen.
Immer öffnen
Immer die URL in einer anderen Anwendung öffnen.
teilen
Immer URLs teilen.
auswählen
Jedes Mal, wenn Sie eine Download-Anfrage erhalten, können Sie auswählen, welches Verfahren (öffnen, herunterladen und teilen) Sie verwenden möchten.

Sucheinstellungen
Suchmaschineneinstellungen
Wählen Sie die von Ihnen eingerichtete Suchmaschine aus.
URL-Einstellungen der Suchmaschine
Sie können den Titel, die URL und die Farbe der Suchmaschine einstellen.
Setzen Sie den Suchbegriff als '%s'
Umschalte-Symbol in der Suchleiste anzeigen
Sie können einstellen, ob das Symbol für die Suchmaschinenumschaltung in der Suchleiste angezeigt werden soll.
Suchvorschlag-Typ
Der Suchvorschlag kann aus dem Netzwerk, aus früheren Abfragen und aus beiden eingestellt werden.
Verlauf anzeigen
Zeigt den Verlauf im Suchvorschlag an.
Lesezeichen anzeigen
Zeigt Lesezeichen im Suchvorschlag an.
Reiter speichern beim Beenden

Sie können einstellen, ob bei Verwendung der Aktion [Beenden] der Reiter gespeichert werden soll.

Angepinnte Reiter beim Beenden speichern

Speichert den angepinnten Reiter bei Verwendung der Aktion [Beenden].

Geschlossene Reiter speichern

Aktivieren Sie diese Option, wenn Sie die Aktion [Reiter wiederherstellen] verwenden
Wenn die Aktion [Reiter wiederherstellen] nicht verwendet wird, ist es besser, die Option zwecks Speicheroptimierung zu deaktivieren.

Einstellungen für Neuer Reiter

Kann Einstellungen beim Öffnen eines neuen Reiters festlegen

Anzahl der Reiter im Zwischenspeicher

Einstellen der Höchstzahl der im Speicher gehaltenen Registerkarten

Web bei Wechsel des Reiters anhalten

Reiter stoppen die Verarbeitung, wenn sie in den Hintergrund treten.

Web im Hintergrund anhalten

Alle Reiter stoppen die Verarbeitung, wenn die Anwendung in den Hintergrund tritt.

Unbekanntes Schema teilen

Versuchen Sie, die URL von einer anderen Anwendung zu öffnen, die kein Schema unterstützt. (z.B. Intent: // oder Twitter: //)

Verlangsamter Bildaufbau
