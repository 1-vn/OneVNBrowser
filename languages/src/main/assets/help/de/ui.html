Einstellungen der Benutzeroberfläche

Design-Einstellungen

Sie können das integrierte und Ihr eigenes Design wählen
Sie können auch das Design des Mikan-Browsers verwenden
Das benutzerdefinierte Design befindet sich im internen Speicher
Beispiel (Bitte entsprechende Änderungen vornehmen):/sdcard/YuzuBrowser/theme/**theme name**/theme.json
Bitte entsprechend so installieren, dass es für den Browser zugriffsfähig ist.

Einstellungen der Symbolleiste

Einstellungen der Schriftgröße

Sie können die Schriftgröße von Lesezeichen, Verlauf, Menü, Menü und der Später-Lesen-Liste ändern.

Touch-Scrollbar-Einstellungen

Zeigt eine Bildlaufleiste an, die Sie am Rand des Browsers berühren können. Sie können einstellen, ob die Symbolleiste beim Scrollen mit der Bildlaufleiste verschoben werden soll.
