Toolbar settings

Snap Toolbar

Upon a scroll ending, if the toolbar is only partially visible then it will be snapped and scrolled to it's closest edge.
For example, if the toolbar only has it's bottom 25% displayed, it will be scrolled off screen completely. Conversely, if it's bottom 75% is visible then it will be scrolled fully into toolbar.
