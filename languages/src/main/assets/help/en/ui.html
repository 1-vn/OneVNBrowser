Toolbar settings

Font size settings

You can change the font size of bookmarks, history, menu, and read it later lists.

Touch scrollbar settings

Displays a scroll bar that you can touch at the edge of the browser. You can set whether to move the toolbar when scrolling with the scroll bar.

Orientation

You can set the orientation of browser screen
Unspecified

Landscape
Force in landscape orientation
Portrait
Force in portrait orientation
Sensor
Detect with the sensor and change the direction
No sensor
Fix the orientation of the screen of the application regardless of the change of orientation of the terminal by the sensor.
