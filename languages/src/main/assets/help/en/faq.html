Frequently Asked Questions

I want to open a certain site and open it with another application.

It can be set by Pattern match.

The font is too big.

You can use Minimum font size setting. If it is still big, use Layout algorithmsetting.

Don't ask on exit.

Finish action→Options (gear button)→Uncheck the "show alert"

For standard settings, there is the Finish action in the following location
[back] action → options → [Distinction tab finish] options → default → [finish] action
[Tab action] settings → [tap] → [Close tab] action → options → [finish] action

Images are not displayed.

Privacy settings→Security settings→Insecure content setting on https

I can not log in to ○○

You may not be able to log in if cookies and/or third party cookies are invalid

I want to change the order of ○○

There is a sort on the option in the upper right.
Example: bookmarks, menu settings, etc.

Can you do ○○ with link and/or image long press?

You can add actions from Action settings→ Long press settings

Do you have a ○ ○ button? How do I put a button?

Action settings→Software button settings→○○ button→Add with the button on the lower right
You can set what is inthe actions list.

What is the "gear" next to the action?

It is an option button. You can change the behavior.

I want to delete ads

Browser settings→Ad block settings

I want to save it on SD card!!!!!!

Only the following path can be saved.
/storage/sdcard1/Android/data/jp.hazuki.yuzubrowser/files

I want to prevent moving even if I scroll the XX bar (tool bar) on the screen.

UI settings→Toolbar settings→XX bar→Position→Choose "XX(Always display)"

I want to show toast when using a action.

You can use Show toast action

I changed the speed dial icon but it does not change.

It will be changed from the next new tab.
Or change it by entering URL and moving (including bookmark etc).
Note: It does not change with page update.
