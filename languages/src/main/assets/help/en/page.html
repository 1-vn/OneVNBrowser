Page settings

Load with override mode

Adjust so that the entire page is displayed

Use wide viewport

Enable HTML viewport or wide viewport
the page contains the viewport meta tag, the value of the width specified in the tag is used.

Enable popup window

Allow popup of dialog displayed in Java Script

Display zoom buttons

Display zoom button on page

Block images

Block images on web page

Fastback

Moving at high speed when going back and forward on the page
It consumes a lot of memory

Fastback cache size

Change the cache size of fast back
0 = unlimited
The more memory you consume the more

Pull to refresh

Pull further at the top of the page to update
It can not be used on some pages

Rendering mode

Change the display color of web page

Minimum font size

Change the minimum font size of web page
*Point If the size is big even 1px, set Layout algorithm setting to "Normal"

Text size

You can change the character size based on 100%

Layout algorithm

You can change the layout algorithm

Night mode settings

You can change the color temperature and brightness when the layout algorithm is in night mode.
