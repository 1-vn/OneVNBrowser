常见问题解答

我想使用其他应用打开特定网站。

可以通过执行模式匹配实现。

字体太大了。

可设置最小字体大小。若字体依然太大，可设置布局算法。

退出时不要询问我。

结束 行为 → 选项（齿轮图标按钮） → 取消勾选“显示警告”

若要设置所有的退出标准，可在以下位置找到结束行为
[返回] 行为 → 选项 → [根据标签页结束] 选项 → 默认 → [结束] 行为
[标签页行为] 设置 → [点击] → [关闭标签页] 行为 → 选项 → [结束] 行为

图像不显示。

隐私设置 → 安全设置 →https 中的不安全内容

无法登录至 ○○

不使用 Cookie 或第三方 Cookie 的情况下可能无法登录

我想更改 ○○ 的顺序

选项界面的右上角通常有排序按钮。
例如：书签、菜单设置等。

可以在长按链接或图像时执行 ○○ 吗？

可在行为设置 → 长按设置中添加行为

有 ○○ 按钮吗？我该怎么放置按钮？

行为设置 → 软件按键设置 → ○○ 按钮→ 通过右下角的按钮添加
可以选择行为列表中的行为。

行为旁边的“齿轮图标”是什么东西？

这是选项按钮，可以更改具体的执行行为。

我想移除广告

浏览器设置 → 广告拦截设置

我要把它保存到存储卡上！现在立刻！！！

仅能保存到以下路径。
/storage/sdcard1/Android/data/jp.hazuki.yuzubrowser/files

我希望在滚动时依然显示 XX 栏（工具栏）。

用户界面设置 → 工具栏设置 → XX 栏 → 位置 → 选择“XX（始终显示）”

我想在执行行为后显示一个提示框。

可以使用显示提示框行为

我更改了快捷访问的图标，但是没有生效。

在下一个新标签页中就会生效了。
你也可以通过输入地址或移动至其他窗口（书签等）来刷新。
注意：页面更新时不会发生变化。
