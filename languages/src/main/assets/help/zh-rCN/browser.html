UserScript 设置

可管理 UserScript

广告拦截设置

可设定广告拦截。
关于如何编写列表的相关信息请查看执行模式说明。

资源拦截设置

根据地址对图像、外部脚本等内容进行拦截。

主页

在开启新标签页时打开指定地址。
默认为快捷访问（yuzu:speeddial）

下载行为

可更改下载文件等内容时的行为。

不进行操作
忽略下载请求。
自动
当请求为视频播放等类似情景时，通过其他应用打开地址，否则显示下载对话窗口。
始终下载
总是显示下载对话窗口。
始终打开
总是在其他应用中打开地址。
分享
总是分享地址。
手动选择
每当收到下载请求时，手动选择执行打开、下载或分享等操作。

代理设置

可设定代理

搜索设置
搜索引擎设置
选择已设定的搜索引擎。
搜索引擎地址设置
可设定搜索引擎的标题、地址和颜色。
在地址中用“%s”规定搜索关键词
搜索栏显示切换图标
可设定是否在搜索栏中显示搜索引擎切换图标。
搜索建议类型
搜索建议可设定为网络来源、过去记录或同时显示两者。
显示历史记录
在搜索建议中显示历史记录。
显示书签
在搜索建议中显示书签。
退出时保存标签页

可设定是否在 [结束] 时保存标签页

退出时保存固定标签页

[结束] 时保存固定标签页

保存关闭的标签页

若要使用 [恢复标签页] 行为请启用此选项
不使用 [恢复标签页] 则建议关闭，以节省内存

新建标签页设置

可设定打开新标签页的设置

缓存标签页数量

设定保存在内存中的标签页的数量上限

切换标签页暂停网页

在标签页移至后台后停止处理

后台暂停网页

分享未知跳转协议

尝试通过不支持跳转协议的其他应用打开地址。（例如 intent:// 或 twitter:// 等）

下载目录

指定下载的目标文件夹
仅限存储卡 /storage/sdcard1/Android/data/jp.hazuki.yuzubrowser/files

自动保存间隔（秒）

设定自动保存间隔（秒）
设为 0 秒则不执行自动保存。

缓慢渲染

用于进行全屏截图的设置。
不建议在其他情况下启用，页面可能会频繁卡死。
