ページ設定

ページを全体表示で開く

ページの全体を表示するように調整する。

ページを常に広域表示にする

HTMLのviewportまたはwide viewportを有効にする
ページにビューポートのメタタグが含まれている場合、タグで指定された幅の値が使用されます。

ページのポップアップを許可する

Java Scriptで表示されるダイアログのポップアップを許可するか変更します

ズームボタンの表示

ページ上にズームボタンを表示します

画像を読み込まない

Webページ上の画像をブロックします

ファストバック

ページを戻る/進むをするときに高速で移動します
メモリを多く消費します

ファストバックキャッシュサイズ

ファストバックのキャッシュサイズを変更します
0で全てをキャッシュ
多いほどメモリを多く消費します

引っ張って更新

ページの先頭で更に引っ張ると更新します
一部のページでは使用できません

レンダリングモード

webページの表示色を変更します

最小フォントサイズ

webページの最小フォントサイズを変更します
※ポイント　サイズを1にしても大きい場合、レイアウトアルゴリズムをNormalにします

フォントサイズ

文字の大きさを100％を基準に変更できます

レイアウトアルゴリズム

レイアウトアルゴリズムを変更できます

ナイトモード設定

レイアウトアルゴリズムがナイトモードの時の色温度と明るさを変更できます。
